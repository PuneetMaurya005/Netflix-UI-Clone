# 🎬 Netflix UI Clone

A pixel-close clone of the Netflix India landing page built from scratch 
using pure HTML, CSS, and JavaScript. No frameworks, no libraries.

## ✨ Features
- Responsive Navbar with Sign Up button
- Hero section with email CTA
- TV section with autoplay video overlay
- Horizontally scrollable Top 10 Movies slider
- Interactive FAQ accordion (open/close with animation)
- Netflix-style footer with language selector

## 🛠️ Built With
- HTML5
- CSS3
- Vanilla JavaScript

2. Open `netflix.html` in your browser
3. That's it!
